Capítulo 5 Análisis de Contexto Formal

Ya que la función para buscar implicaciones con este dataset tarda demasiado, vamos a quitar todas las variables que estén muy relacionadas entre sí y vamos a crear un contexto formal con las variables restantes

correlationMatrix <- cor(Emple01)
print(correlationMatrix)
relacionado <- findCorrelation(correlationMatrix, cutoff=0.5)
print(relacionado)


fc_Empleo <- FormalContext$new(Emple01[-relacionado])

fc_Empleo$plot()

Clarificamos (para quitar duplicados) y reducimos (auto-explicativo)

fc_Empleo$clarify(TRUE)
fc_Empleo$reduce(TRUE)

fc_Empleo$find_concepts()
