2.2.2 Job.search.problems.grouped

Veamos primeros cuántos datos diferentes hay

unique(unlist(Empleo$Job.search.problems.grouped))

Hay 16 diferentes tipos de valores, vamos a intentar siempre que se queden en lo mínimo (3-5 a poder ser)

for (x in c("No problems ","No problems",
            "Still not looking for jobs actively",
            "I haven't applied for any. ","Nothing ",
            " interviewer not handsome ", "Good oppurtunities")
    ) {
  Empleo$Job.search.problems.grouped <- gsub(x,"Nothing",
                                             Empleo$Job.search.problems.grouped)
}

for (x in c("Lack of experience","Fear of English"," network slow at home")) {
  Empleo$Job.search.problems.grouped <- gsub(x,"My thing",
                                             Empleo$Job.search.problems.grouped)
}

Me gustaría indicar, por si no os habéis dado cuenta de que alguien como respuesta a los problemas encontrados buscando trabajo ha puesto: “interviewer not handsome” ノಠ_ಠノ

¿Para que sirve la función “gsub”? Os podríais preguntar. El asunto es que Las respuestas nos vienen dadas como strings del estilo “A;B;C,D” no como c(A,B,C,D) por lo que la función coje dos cadenas de tal forma que una servirá de patrón que si es encontrado será reemplazado en el string por el contenido de la segunda cadena
